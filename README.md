# RevertReaperNerf for Zumbi Blocks 2

RevertReaperNerf is a server-side mod for [Zumbi Blocks 2](https://store.steampowered.com/app/1941780/Zumbi_Blocks_2_Open_Alpha/) that reverts nerfs to the Reaper made in version **2.1.2.1**.

## Details

- **Gun Damage**
  - Original: 4
  - Nerfed: 3
- **Stagger Chance**
  - Original: 100%
  - Nerfed: 50%
- **Action Count**
  - Original: 5
  - Nerfed: 4
- **Action Cooldown**
  - Original: 0.1 - 0.2 for all attacks
  - Nerfed: 0.25 - 0.5 for regular attacks, 3 - 6 for "heavy" attacks
